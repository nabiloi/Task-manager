# CLI Task Manager
  CLI Task Manager with Data Export Capability
A practical Python-based Command-Line Interface (CLI) application designed to manage daily tasks efficiently. 

This project reflects my early journey in self-taught programming, focusing on core logic and data portability. 

## 🚀 Key Features
 * Task Management: Create, view, mark as complete, and delete tasks seamlessly via interactive menu.
 * Data Persistence: Automatically saves tasks to a .txt file to ensure no data loss.
 * Multi-format Export: Built-in functionality to export task lists into .json and .csv formats (compatible with Excel).
 * Command Handling: Supports both interactive input and direct terminal arguments (e.g., python "Task manager.py" add "Task Name").
 * Error Prevention: Basic error handling for invalid IDs and file management.

## 🛠️ Tech Stack
 * Language: Python 3.x
 * Standard Libraries: os, sys, json, csv

## 📖 How to Use
 * Clone this repository.
 * Run the application:
 * * ```
     python "Task manager.py"
     ```

 * Use --help for command-line shortcuts or follow the on-screen menu.
